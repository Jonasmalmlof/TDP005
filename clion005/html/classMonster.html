void Monster::move ( )

Moves the Monster to the next step of its patrol route unless the player intersects with the Monster's sight box. The Monster's sight box reaches one time the Monster object's width in all 8 directions.

Step is the variable for which part of the patrol the monster is on, if the monster has completed its route step is set to 0 and the patrol resets
